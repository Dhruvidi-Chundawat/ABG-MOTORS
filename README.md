# Market Entry Analysis for ABG Motors (India)

## Project Overview
ABG Motors, a Japanese automobile manufacturer, is considering entering the Indian market. 
This project uses data science techniques to predict potential car buyers in India based on 
patterns learned from the Japanese market.

## Objective
- Build a classification model to predict car purchase behavior
- Estimate potential customers in India
- Support market entry decision with data-driven insights

## Tools & Technologies
- Python
- Pandas, NumPy, Scikit-learn
- Logistic Regression
- Tableau
- Google Colab

## Methodology
1. Data Cleaning & Preprocessing
2. Feature Engineering
3. Logistic Regression Model
4. Model Evaluation (Accuracy, ROC-AUC)
5. Indian Market Prediction
6. Tableau Visualization

## Key Outcome
The model predicts that the Indian market has sufficient potential customers to meet the 
minimum sales target, supporting a positive market entry decision.
